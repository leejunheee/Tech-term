### 시노님
  - 쉽게 말해 Alias 같이 이름을 줄여주는 역할
  - 테이블의 이름을 설정 
  - 보통 다른 유저의 객체(테이블, 뷰, 프로시저, 함수, 패키지, 시퀀스 등)를 참조할 때 많이 사용 
  - 시노님 생성 / 삭제 
    - CREATE OR REPLACE SYNONYM [시노님명] FOR [객체 이름] 
    - DROP SYNONYM [시노님명]

### 엔티티와 레코드 차이 
  - 레코드 : 실제 데이터베이스 상에 저장되어 있는 값들의 모임
  - 엔티티 : 현실 세계에 존재하는 객체를 표현하기 위해 비유(추상)적으로 사용
